# Captcha Protect Docs

This repository contains the documentation site for
[libops/captcha-protect](https://github.com/libops/captcha-protect).

Published docs: <https://captcha-protect.libops.io/>

## Local preview

```bash
make docs-serve
```

`docs-serve` runs a live-reload preview at <http://localhost:8888>.

Other useful targets:

```bash
make docs-build
make docs-preview
make docs-clean
```
